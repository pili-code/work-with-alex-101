# Work with Alex 101

How to prep for and run GTM Syncs (and any update) with Alex (Alessandro Alter), TDP cofounder.

Alex is business- and numbers-oriented. He wants the conclusion first, a number on every claim, and a decision to make. This repo packages that into a reusable Claude skill plus the source analysis.

## Contents

- `.claude/skills/work-with-alex/SKILL.md` — the reusable playbook skill. Triggers on GTM Sync prep, agenda building, or "how do I present this to Alex".
- `analysis/gtm-sync-analysis.md` — the source analysis of what Alex actually asks, pulled from GTM Sync transcripts (Fireflies).
- `analysis/agenda-template.md` — the improved agenda structure.

## TL;DR — how to talk to Alex

1. Lead with the conclusion. Never narrate the process.
2. Every claim gets a number (conversion rate, $, count).
3. Separate paying customers from everyone else.
4. Each deal: name → status → why → next action → owner.
5. Flag each item: Decision / FYI / Blocked. Spend time only on Decision + Blocked.
6. End with "what I need from you."
